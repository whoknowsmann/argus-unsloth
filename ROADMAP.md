# Obsidian Reborn - Core Feature Roadmap

## Missing Core Features

- [ ] **Tags** — Parse `#tag` syntax, index tags, add tag filter/search
- [ ] **Embeds** — Render `![[note]]` content inline (not just images)
- [ ] **Global Graph View** — Full vault visualization (currently only local/neighbor graph)
- [ ] **Frontmatter/YAML** — Parse and display `---` metadata blocks
- [ ] **Outline Panel** — Extract headers (H1-H6) as clickable TOC for current note
- [ ] **Starred/Bookmarks** — Pin favorite notes for quick access
- [ ] **Templates** — Insert predefined content snippets into notes
- [ ] **Callouts (full)** — Expand beyond basic `> [!TYPE]` support

## Stretch Goals

- [ ] Hover preview on wikilinks
- [ ] PDF/HTML export
- [ ] Vim keybindings (CodeMirror has a plugin)
- [ ] Slash commands (`/template`, `/date`, etc.)
- [ ] Note history/versioning
- [ ] Drag-and-drop in file tree
- [ ] Image paste from clipboard
- [ ] Mermaid diagram rendering
